# JewelHickman.github.io

Index page:
Options to make an account, search for other accounts

Home page:
User can view / access their posts, view saved posts, make new post, search bar

Make blog post:
Text input w/ preview button

Userpage
All of a user's posts are displayed in chronological order (most recent first)

Blog post
View a specific blog post. Option to comment, like/save
